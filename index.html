In this kata, you must define the Array.reduce method.
The reduce method goes through each element of an array, applies the function to whatever the function returned last, and returns the last outcome. On the first iteration, it should pass the initial value to the function, as well as the first element of the array. If no initial value is passed, skip to the first element of the array. 

          
            Array.prototype.reduce = function(process, initial) {
              let res;
              if(initial === undefined) {
                (typeof this[0] === 'number') ? res = 0 : res = '';
              } else {
                res = initial;
              }
              
              for (let i = 0; i < this.length; i += 1) {
                res = process(res, this[i], i, this);
              }
              return res;
            }
